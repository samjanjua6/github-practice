# Intermediate Python Learning Repo

This repository is a practice space for learning intermediate Python concepts through small scripts, examples, and experiments.

## Goal

The goal of this repo is to move beyond basic syntax and build confidence with the Python features used in real projects.

## Topics to Practice

- Functions and scope
- Tuples, lists, dictionaries, and sets
- File handling
- Error handling
- Modules and imports
- Object-oriented programming
- Iteration and comprehensions
- Working with Python standard library tools

## Files

- python.py - general Python practice and examples
- tuples.py - tuple-related exercises and notes

## How to Use This Repo

1. Open the Python file you want to practice with.
2. Run the script and observe the output.
3. Modify the code, test ideas, and keep notes on what you learn.
4. Add new files as you explore more Python topics.

## Learning Approach

I will use this repo to:

- write small examples
- try out new concepts
- make mistakes and fix them
- build a stronger understanding of Python step by step
